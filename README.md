# 🌍 Village Digital Twin AI

> AI-Powered Rural Intelligence Platform for Sustainable Development

Village Digital Twin AI is an AI-powered decision support system that predicts village-level environmental and disaster risks using Machine Learning, GIS mapping, analytics, and interactive dashboards.

Built for **IDEA2IMPACT Hackathon 2026**.

# ⚙ Installation

Clone

```bash
git clone https://github.com/Neeraja-Pyla/village-digital-twin-ai.git
```

Frontend

```bash
cd frontend
npm install
npm run dev
```

Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```
